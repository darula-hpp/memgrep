# Memory Tools

Typical tools (names may grow across releases):

- `recall` - search the local store
- `remember` - store a note / playbook
- `get_chat` - fetch a chat by id
- `list_chats` / related listing tools

Agents should prefer `recall` before inventing a procedure that might already be stored.
