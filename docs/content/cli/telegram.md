# Telegram CLI

```bash
memgrep telegram                 # interactive / foreground poller
memgrep telegram install [--all]
memgrep telegram service
memgrep telegram uninstall
```

Profiles live under `~/.memgrep/`. Use `--all` to install every configured bot profile as LaunchAgents.
